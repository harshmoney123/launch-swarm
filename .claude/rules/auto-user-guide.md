# Rule 14: Auto User Guide

**Owner: Docs agent** (`loop-docs.md`). Blog drafts are part of the Documented gate.

## Trigger

`feat:` PRs in your frontend repo with screenshots already captured.

## Template for Content Tool

```
Title: "How to [feature action]"
Category: User Guide

1. **What's New** -- [1-2 sentence summary]
2. **Step-by-Step Guide** -- [numbered steps, screenshot after EVERY step]
3. **Tips** -- [edge cases, power-user notes]

Screenshots attached via media_urls. Write for a new user. DRAFT only.
Context: PR [URL], Notion [URL]
```

Create via your content automation tool (e.g., a CMS session API or AI content engine).
Link session ID on PR comment + Notion task.
